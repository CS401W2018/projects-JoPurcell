Engagement Questions

  1. What is the difference between block tags and inline tags.

    2. Block tags are tags that will create their own space. THey have a sort of invisible box surrounding them. Inline tags are tags that only take up the necessary width it feels it should take.

  2. What is the margin, border, and padding?

    3. These are tags used to edit text in a block or box. The margin is the space around the whole text/block. Margins pushes the box away from other blocks/boxes. Border is a visible box that contains all the content. THe padding is the space between content

  3. What are units of measurement you can use in CSS?

    4. Pixels is the most popular. We can use milimeters and percentages, too.

  4. When would you use a figure tag?

    5. When we want to feature something on our web page with it's own little segment.
