Engagement Questions

  1. The form tag has an "action" attribute. What does it do?

    2. Action attribute can help us submit the form to the server.

  2. The form tag has a "method" attribute. What does it do?

    3. Can allow us to send data over the server.

  3. What is the difference between a "POST" and a "GET"?

    4. All information from form goes up to the search bar in the query string when we use a GET attribute. No information from our form tag is shown in the query string for POST attribute.
