Engagement Questions

  1. Explain the difference between inline, embedded, and external CSS.

    2. Inline CSS is added in the HTML attribute in the HTML page. Embedded is similar, but the selectors are added in the head tag of the HTML. External HTML is a separate stylesheet file. This allows the HTML attributes to overwrite it.

  2. Explain why you should only use external CSS.

    3. We should use external CSS as it will be easier to edit without changing the entire HTML and possibly ruining it. Using external CSS is neater and more organized too.

  3. What is a selector used for?

    4. A selector is used to select specific tags we want to format in the CSS stylesheet.
