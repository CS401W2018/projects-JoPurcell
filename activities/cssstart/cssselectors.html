Engagement Questions

  1. List out the different ways to select an item

    2. Use tag name, IDs, Class and descendant

  2. What is the difference between an ID and a Class?

    3. IDs should be unique and should not be used more than once. They have limitations. When used in CSS, we use hashtag/pound symbols. For a class, we can use this more than once. A tag can have more than one class. We use a period in when selecting it in the CSS. Classes can be applied to any tag we want unlike IDs.

  3. What is the difference between a SPAN tag and a DIV tag?

    4. A SPAN tag is used to select specific text and applies changes to the selected text. DIV tag takes a specific section of our code and creates it's own block/box.

  4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

    5. DIV collects the whole chunk of our code and puts it in its own separate block. Header, main and footer are tags with content. They can be in the DIV tag. In comparison to DIV tag, they don't create a big breaking space.

  5. What are different ways to select a color?

    6. Selecting a predefined color or selecting a custom color with the color picker.
